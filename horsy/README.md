### Installation

```bash
# YOLO
curl -fsSL https://agenthorsy.ai/install | bash

# Package managers
npm i -g agenthorsy-ai@latest        # or bun/pnpm/yarn
scoop install agenthorsy             # Windows
choco install agenthorsy             # Windows
brew install anomalyco/tap/agenthorsy # macOS and Linux (recommended, always up to date)
brew install agenthorsy              # macOS and Linux (official brew formula, updated less)
sudo pacman -S agenthorsy            # Arch Linux (Stable)
paru -S agenthorsy-bin               # Arch Linux (Latest from AUR)
mise use -g agenthorsy               # Any OS
nix run nixpkgs#agenthorsy           # or github:anomalyco/agenthorsy for latest dev branch
```

> [!TIP]
> Remove versions older than 0.1.x before installing.

### Desktop App (BETA)

Horsy is also available as a desktop application. Download directly from the [releases page](https://github.com/anomalyco/agenthorsy/releases) or [agenthorsy.ai/download](https://agenthorsy.ai/download).

| Platform              | Download                           |
| --------------------- | ---------------------------------- |
| macOS (Apple Silicon) | `agenthorsy-desktop-mac-arm64.dmg`   |
| macOS (Intel)         | `agenthorsy-desktop-mac-x64.dmg`     |
| Windows               | `agenthorsy-desktop-windows-x64.exe` |
| Linux                 | `.deb`, `.rpm`, or `.AppImage`     |

```bash
# macOS (Homebrew)
brew install --cask agenthorsy-desktop
# Windows (Scoop)
scoop bucket add extras; scoop install extras/agenthorsy-desktop
```

#### Installation Directory

The install script respects the following priority order for the installation path:

1. `$AGENTHORSY_INSTALL_DIR` - Custom installation directory
2. `$XDG_BIN_DIR` - XDG Base Directory Specification compliant path
3. `$HOME/bin` - Standard user binary directory (if it exists or can be created)
4. `$HOME/.agenthorsy/bin` - Default fallback

```bash
# Examples
AGENTHORSY_INSTALL_DIR=/usr/local/bin curl -fsSL https://agenthorsy.ai/install | bash
XDG_BIN_DIR=$HOME/.local/bin curl -fsSL https://agenthorsy.ai/install | bash
```

### Agents

Horsy includes two built-in agents you can switch between with the `Tab` key.

- **build** - Default, full-access agent for development work
- **plan** - Read-only agent for analysis and code exploration
  - Denies file edits by default
  - Asks permission before running bash commands
  - Ideal for exploring unfamiliar codebases or planning changes

Also included is a **general** subagent for complex searches and multistep tasks.
This is used internally and can be invoked using `@general` in messages.

Learn more about [agents](https://agenthorsy.ai/docs/agents).
